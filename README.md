# Hearo - Sound Classification Project

## Overview
This project aims to build a sound classification model using the ESC-50 dataset.

## Dataset
Used Dataset: [ESC-50](https://github.com/karolpiczak/ESC-50)
- 50 classes of environmental sounds
- 2000 clips (5 seconds each)

## Project Structure
- `data/`: Raw and processed data
- `src/`: Source code
- `notebooks/`: Jupyter notebooks for analysis
- `tests/`: Unit tests
